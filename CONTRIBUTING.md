# Guia de Contribuição

Obrigado por considerar contribuir para o Sistema de Sorteio JR!

## Como Contribuir

### 1. Fork e Clone
```bash
git clone https://github.com/seu-usuario/jr-sorteio.git
cd jr-sorteio
```

### 2. Crie uma Branch
```bash
git checkout -b feature/sua-feature
```

### 3. Faça as Mudanças
- Siga a paleta de cores da marca
- Mantenha a estrutura do código
- Adicione comentários quando necessário

### 4. Teste Localmente
```bash
streamlit run app.py
```

### 5. Commit e Push
```bash
git add .
git commit -m "feat: descrição da sua mudança"
git push origin feature/sua-feature
```

### 6. Abra um Pull Request
Descreva as mudanças e referencie issues relacionadas.

## Padrões de Código

- Use Python 3.8+
- Siga PEP 8
- Nomes em inglês para variáveis e funções (comentários em português)
- Docstrings para funções complexas

## Reportar Bugs

Use a seção "Issues" no GitHub com detalhes:
- Descrição clara do bug
- Passos para reproduzir
- Comportamento esperado vs. observado
- Ambiente (Python, Streamlit, SO)

## Sugestões de Melhorias

Todas as sugestões são bem-vindas! Abra uma issue com o prefixo `[SUGESTÃO]`.
